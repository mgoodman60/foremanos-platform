# ForemanOS Verified Features (Truth Table)

## ✅ VERIFIED - Currently Live Features

### Core Platform
- **Document Upload & Storage** - Users can upload project documents (PDF, DOCX, images)
- **AI Chat Interface** - Natural language Q&A based on uploaded documents  
- **Multi-Project Organization** - Create/manage multiple construction projects
- **Document Search** - Find information across all project documents
- **User Authentication** - Secure login with email/password
- **Role-Based Access** - Admin, Client, Guest, and Pending user roles
- **Guest Access Links** - Share project access via Job Pin credentials
- **Document Permissions** - Control who can view each document (admin/client/guest)
- **Activity Logging** - Track user actions and document access
- **Email Notifications** - Automated emails for signups, approvals, document uploads
- **Subscription Management** - Stripe-integrated payment system (6 tiers: Free, Starter, Pro, Team, Business, Enterprise)
- **OneDrive Sync** - Automated document syncing from OneDrive folders

### Intelligence Features  
- **RAG (Retrieval-Augmented Generation)** - AI answers grounded in project documents
- **3-Tier Model Routing** - GPT-3.5, Claude 3.5, GPT-4o based on query complexity
- **OCR/Vision Processing** - Extract text and metadata from construction plans
- **Spatial Metadata Extraction** - Identify sheet numbers, scales, dimensions from plans
- **Query Caching** - Reduce costs by caching common queries
- **Web Search Integration** - Supplement document answers with web sources when helpful

### User Experience
- **Project Dashboard** - View all projects and document counts
- **Document Library** - Browse, search, filter uploaded documents
- **Conversation History** - Review past AI chat conversations  
- **Feedback System** - Thumbs up/down on AI responses
- **Progress Documentation** - Activity feed of project updates

### Admin Features
- **User Approval System** - Admins approve pending user signups
- **Analytics Dashboard** - Track usage, costs, and user activity
- **Cost Monitoring** - Real-time tracking of AI query costs
- **Activity Logs** - Audit trail of all system actions

### Security & Compliance
- **Encrypted Storage** - All documents stored securely in AWS S3
- **Session Management** - Secure authentication via NextAuth
- **Rate Limiting** - Prevent abuse on critical endpoints
- **Audit Trail** - Activity logging for compliance

## ❌ NOT VERIFIED - Do Not Mention

The following features do NOT currently exist and should NOT be mentioned:

- GPS tracking / location services
- Time tracking or timesheets
- Equipment management
- Invoicing or billing
- RFI (Request for Information) management
- Punch list management
- BIM/3D model viewing
- Procore, Buildertrend, or other third-party integrations (except OneDrive)
- Mobile native apps (iOS/Android)
- Offline mode
- Video calls or real-time collaboration tools
- Calendar scheduling
- Photo/video AI processing (not yet implemented)
- Progress photos with AI analysis
- Safety monitoring via computer vision
- Material delivery tracking
- Automated reporting beyond what's in admin dashboard

## 🎯 Positioning Statement (Truth-Based)

**ForemanOS is a document-based AI assistant for construction teams.**

It helps you:
- Upload project documents in one place
- Ask questions and get instant answers from those documents  
- Organize multiple projects with team access controls
- Track document access and activity
- Connect OneDrive for automatic syncing

That's it. No scheduling, no time tracking, no invoicing. Just accurate, document-based answers when you need them.
